# Codex_pretest

このリポジトリには、Tkinter で実装したシンプルなテトリスが含まれています。

## 動作方法

1. Python 3 がインストールされていることを確認します（Tkinter は標準で含まれています）。
2. ルートディレクトリで次のコマンドを実行します。

   ```bash
   python tetris_gui.py
   ```

   GUI ウィンドウが開き、テトリスをプレイできます。

## 操作方法

- ← / →: ブロックを左右に移動
- ↑: ブロックを回転
- ↓: ソフトドロップ
- スペース: ハードドロップ
- P: 一時停止
- R: ゲームオーバー後に再スタート
